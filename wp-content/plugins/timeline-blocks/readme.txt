A beautiful timeline layout block to showcase your posts in timeline presentation.

== Description ==

Timeline Blocks for Gutenberg is a Gutenberg add-on with a bunch of additional options for post showcase with vertical timeline. This is a standalone Gutenberg solution to give wow design possibility for your blog posts with clean design and full responsive layout.

You can say that **Timeline Blocks** will be your *Ultimate History Showcase* for new Gutenberg Editor.

= Features of Timeline Blocks for Gutenberg =

* 2 Timeline Template
* Option to show/hide multiple post meta
* Easy to install, use and customize
* Clean code, layout and design
* Fully responsive for any devices
* 'Read More' text option to choose and change
* Order by - Newest to Oldest, Oldest to Newest, A → Z, Z → A
* Featured Image Style - Landscape, Square

= Why choose Timeline Blocks for your Gutenberg Project? =

We carefully researched over minimum requirements of any type of history showcase presentation. And then created list of options to provide that flexibility to new Gutenberg users. Yes, it's free for everyone and you can see the instant result with the Gutenberg Editor while updating your changes with the options. We provided all possible options with this `Timeline Blocks for Gutenberg` plugin. So, from beginner user of Gutenberg to any kind of experts, bloggers and freelancers can use this standalone solution.

= Where this Timeline Blocks useful? =

* Showcase yours posts with timeline view in any post and page
* Showcase your company growth day by day (or month by month, or year by year)
* Showcase your projects that you completed in past with dates
* Showcase step by step tutorial with post title like 'Story #1, Podcast #1, etc.'


== Installation ==

1. Upload the `timeline-blocks` folder to your `/wp-content/plugins/` directory or alternatively upload the timeline-blocks.zip file via the plugin page of WordPress by clicking 'Add New' and selecting the zip from your computer.
2. Install and activate the Gutenberg WordPress plugin.
3. Activate the `Timeline Blocks` WordPress plugin through the 'Plugins' menu in WordPress.
4. Use **Timeline Blocks** blocks from 'Timeline Blocks by Techeshta' Gutenberg block registered category on your next page or post.
